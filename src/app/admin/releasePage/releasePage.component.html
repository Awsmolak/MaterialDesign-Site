Release Management

For a release to go out the list below needs to be tested.

  • Assign Codepoints
  • Generate @mdi/svg
  • Verify @mdi/svg works with the webfont builder
  • Generate @mdi/font
  • Update @mdi/svg and @mdi/font repos.
  • @mdi/js uses npm run build
  • @mdi/angular-material uses npm run build
